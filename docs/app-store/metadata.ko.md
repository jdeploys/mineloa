### 설명

Mineloa는 회의를 녹음하고 전사·요약해 주는 독립 실행형 macOS 앱입니다. Mineloa 계정이나 별도 서버 없이 바로 시작할 수 있으며, 회의 기록은 기본적으로 이 Mac에 저장됩니다.

주요 기능

• 회의 녹음, 일시정지, 재개 및 안전한 복구
• OpenAI를 이용한 화자 분리 전사와 구조화 요약
• 다운로드한 Whisper 모델을 이용한 기기 내 로컬 전사
• 원하는 항목과 순서로 구성하는 요약 템플릿
• 화자 이름 수정과 회의별 원본 오디오 보존 정책
• Mineloa 기록 파일 가져오기·내보내기 및 Markdown 내보내기
• 라이트·다크 테마

OpenAI 기능은 사용자가 직접 입력한 OpenAI API 키를 사용합니다. 선택한 처리 방식에 필요한 데이터만 전송되며, 로컬 Whisper 전사는 오디오를 외부 AI 서비스로 전송하지 않습니다. API 키는 macOS Keychain에 저장됩니다.

로그인이나 API 키 없이도 녹음과 로컬 기록 관리를 사용할 수 있습니다. 네트워크 오류나 AI 처리 실패가 발생해도 원본 녹음은 안전하게 유지됩니다.

### 심사 메모

Mineloa 계정 로그인은 필요하지 않습니다.

기본 검토 절차:
1. 앱을 실행하고 마이크 접근을 허용합니다.
2. “녹음 시작”을 누른 뒤 짧게 말하고 녹음을 종료합니다.
3. 녹음과 로컬 기록 관리는 OpenAI API 키 없이 검토할 수 있습니다.
4. OpenAI 전사·요약은 사용자가 자신의 API 키를 설정해야 하므로, 심사자가 별도 키를 입력하지 않아도 앱의 나머지 기능을 검토할 수 있습니다.
5. 로컬 전사를 검토하려면 설정의 고급 처리 옵션에서 Whisper 모델을 다운로드합니다. 모델은 데이터 파일이며 앱 컨테이너에 저장됩니다.
6. 마이크 오디오는 OpenAI 전사를 선택한 경우에만 OpenAI로 전송됩니다. 전사문은 OpenAI 요약을 선택한 경우에만 OpenAI로 전송됩니다.
7. App Store 샌드박스판에서는 외부 Codex CLI 연동을 사용할 수 없습니다.

파일 가져오기와 내보내기는 사용자가 macOS 파일 선택 대화상자에서 명시적으로 고른 파일에만 접근합니다.
