# Mineloa 개인정보 처리방침

최종 업데이트: 2026년 7월 18일

Mineloa는 계정 없이 사용할 수 있는 macOS 회의 기록 앱입니다. 회의, 녹음, 전사문, 요약, 설정과 다운로드한 Whisper 모델은 기본적으로 사용자의 기기에 저장됩니다. Mineloa 개발자는 이 데이터를 운영하는 별도 서버로 수집하지 않습니다.

사용자가 OpenAI 처리를 선택하면 전사를 위해 녹음 오디오가, 요약을 위해 전사문이 사용자가 제공한 OpenAI API 키를 통해 OpenAI API로 전송됩니다. OpenAI의 데이터 처리는 OpenAI의 약관과 개인정보 처리방침을 따릅니다. 로컬 Whisper 전사는 오디오를 외부 AI 서비스로 전송하지 않습니다.

OpenAI API 키는 macOS Keychain에 저장되며 회의 데이터베이스나 내보낸 파일에 포함되지 않습니다. 사용자는 앱에서 API 키와 회의 기록을 삭제할 수 있습니다. 앱 제거 시 운영 체제 정책에 따라 로컬 데이터가 남을 수 있으므로, 필요한 경우 앱에서 먼저 기록을 삭제해야 합니다.

Mineloa는 광고 추적, 제3자 광고 SDK 또는 Mineloa 계정용 분석 식별자를 사용하지 않습니다.
